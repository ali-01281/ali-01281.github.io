About This Project

I created this game to learn Swift and game development. It features:

  • Rabbit character controlled by swiping
  • Maze walls that block movement
  • Carrot goal that wins the game
  • Simple but challenging gameplay

Development Challenges

Challenge: Making the Rabbit Move Smoothly

At first, the movement was jumpy and didn't feel natural.

Solution: Swipe Gesture Recognizers

I used UISwipeGestureRecognizer to detect swipes in four directions. When the player swipes, I move the rabbit one full square in that direction.

Challenge: Wall Collisions

The rabbit could walk through walls, which ruined the game.

Solution: Grid-Based Collision Detection

I created a grid system where each position is either empty or a wall. Before moving, I check if the next position is walkable.

What I Learned

  • How to use Swift for game development
  • Basic collision detection techniques
  • Handling user input with gesture recognizers
  • Problem-solving when things don't work as expected
  • The importance of playtesting
