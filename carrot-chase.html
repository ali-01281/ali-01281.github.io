About This Project

Carrot Chase is an iOS maze navigation game developed using Swift and SwiftUI. The project was designed to demonstrate proficiency in iOS development, SwiftUI implementation, and game logic architecture.

The game features a Rabbit that players guide through a maze to locate it's long lost carrot. This project showcases my ability to create a fun user experiences while implementing game mechanics in a modern iOS development environment.

Through this project, I gained valuable experience in optimizing performance for iOS devices, implementing gesture-based controls, and designing intuitive game interfaces that provide immediate feedback to players.

Development Challenges & Solutions

Smooth Character Movement

Initial implementation resulted in incorrect character animation and inconsistent grid-based movement, creating a poor user experience.

Solution

Used @State variables playerRow and playerCol to store the player’s position and updated them using directional buttons. Each movement button changed the state by incrementing or decrementing the appropriate coordinate.

Wall Collision Detection

The rabbit could initially pass through walls due to insufficient collision checks during the animation phase.

Solution

Created an isValidMove(row:col:) function that checks whether the tile is a valid path or the carrot before allowing the player to move. This function was called before any movement, preventing movement into wall tiles.

Adaptive Layout

Game elements rendered inconsistently across different iOS devices and screen sizes.

Solution

Utilized SwiftUI's GeometryReader and size classes to create responsive layout constraints. Implemented a dynamic grid system that scales elements proportionally while maintaining gameplay integrity.
